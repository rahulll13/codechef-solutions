<p>If Chef's commands have a frequency of <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>X</mi></mrow><annotation encoding="application/x-tex">X</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.07847em;">X</span></span></span></span></span> Hertz, find whether <em>binary</em> can hear them or not.</p>
<div class="notranslate">
<h3>Input Format</h3>
<ul>
<li>The first line of input will contain a single integer <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>T</mi></mrow><annotation encoding="application/x-tex">T</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.13889em;">T</span></span></span></span></span>, denoting the number of test cases.</li>

</ul>
</div><div class="notranslate">
<h3>Output Format</h3>
<p>For each test case, output on a new line <code>YES</code>, if <em>binary</em> can hear Chef's commands. Otherwise, print <code>NO</code>.</p>
<p>The output is case-insensitive. Thus, the strings <code>YES</code>, <code>yes</code>, <code>yeS</code>, and <code>Yes</code> are all considered the same.</p>
