<h2><a href="https://www.codechef.com/practice/course/basic-programming-concepts/DIFF500/problems/SNDMAX?tab=statement">Second Max of Three Numbers</a></h2><h4>Difficulty: 500</h4><div class="_problemStatementWrapper_bh3c4_33"><div class="_problemBodyContent_bh3c4_71"><h3 class="notranslate">Second Max of Three Numbers</h3><h3>Problem Statement</h3>
<p>Write a program that accepts sets of three numbers, and prints the <em>second-maximum number</em> among the three.
</p>
<h3>Input</h3>
<ul><li>First line contains the number of triples, <b>N</b>.</li>
<li>The next <b>N</b> lines which follow each have three space separated integers.</li>
</ul><h3>Output</h3>
<p>For each of the <b>N</b> triples, output one new line which contains the second-maximum integer among the three.</p>
<h3>Constraints</h3>
<ul><li><b>1</b> ≤ <b>N</b> ≤ <b>6</b></li>
<li><b>1</b> ≤ every integer ≤ <b>10000</b></li>
<li>The three integers in a single triplet are all distinct. That is, no two of them are equal.
</li></ul>
