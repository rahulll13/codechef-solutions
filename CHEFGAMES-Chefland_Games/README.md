<h2><a href="https://www.codechef.com/practice/course/logical-problems/DIFF800/problems/CHEFGAMES?tab=statement">Chefland Games</a></h2><h4>Difficulty: N/A</h4><div class="_problemStatementWrapper_bh3c4_33"><div class="_problemBodyContent_bh3c4_71"><h3 class="notranslate">Chefland Games</h3><p>In Chefland, a tennis game involves <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>4</mn></mrow><annotation encoding="application/x-tex">4</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6444em;"></span><span class="mord">4</span></span></span></span></span> referees.<br>

<p>Given the decision of the <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>4</mn></mrow><annotation encoding="application/x-tex">4</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6444em;"></span><span class="mord">4</span></span></span></span></span> referees, help Chef determine whether the ball is considered inside limits or not.</p>

<h3>Output Format</h3>
<p>For each test case, output <code>IN</code> if the ball is considered to be inside limits by all referees and <code>OUT</code> otherwise.</p>
<p>The checker is case-insensitive so answers like <code>in</code>, <code>In</code>, and <code>IN</code> would be considered the same.</p>
