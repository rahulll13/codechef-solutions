<h2><a href="https://www.codechef.com/practice/course/basic-programming-concepts/DIFF500/problems/CANDIVIDE?tab=statement">Candy Division</a></h2><h4>Difficulty: 500</h4><div class="_problemStatementWrapper_bh3c4_33"><div class="_problemBodyContent_bh3c4_71"><h3 class="notranslate">Candy Division</h3><p>There are <strong>three</strong> friends and a total of <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi></mrow><annotation encoding="application/x-tex">N</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.10903em;">N</span></span></span></span></span> candies.<br>
There will be a fight amongst the friends if all of them do <strong>not</strong> get the same number of candies.</p>
<p>Chef wants to divide <strong>all</strong> the candies such that there is <strong>no fight</strong>. Find whether such distribution is possible.</p>
<div class="notranslate">
<h3>Input Format</h3>
<ul>
<li>The first line of input will contain a single integer <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>T</mi></mrow><annotation encoding="application/x-tex">T</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.13889em;">T</span></span></span></span></span>, denoting the number of test cases.</li>
<li>Each test case consists of a single integer <span class="math math-inline"><span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi></mrow><annotation encoding="application/x-tex">N</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.10903em;">N</span></span></span></span></span> - the number of candies.</li>
</ul>
</div><div class="notranslate">
<h3>Output Format</h3>
<p>For each test case, output <code>YES</code>, if we can distribute all the candies between the three friends equally. Otherwise output <code>NO</code>.</p>
<p>You can output each character of the answer in uppercase or lowercase. For example, the strings <code>yEs</code>, <code>yes</code>, <code>Yes</code>, and YES are considered the same.</p>
