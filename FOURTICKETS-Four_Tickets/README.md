<ul>

</ul>

<h3>Output Format</h3>
<p>For each test case, output <code>YES</code> if they will be going to the concert, <code>NO</code> otherwise.</p>
<p>You can print each character in uppercase or lowercase. For example, the strings <code>YES</code>, <code>yes</code>, <code>Yes</code>, and <code>yES</code>, are all considered identical.</p>
